# Tech Stack

## Frontend
- **HTML5:** For structured content and semantic markup.
- **CSS3:** For styling, including modern layout techniques like Flexbox or Grid, adhering to a minimalist aesthetic.
- **Vanilla JavaScript (ES6+):** For application logic, dynamic DOM manipulation, and efficient data filtering without heavy framework overhead.

## Data Management
- **Local Data:** Data is stored in `data.js` as a JavaScript object for high-speed access and nearly instant filtering response.
- **Raw Data Source:** `athlete_events_sample.csv` serves as the primary data source for extraction and transformation.

## Tools & Environment
- **Standard Web Browser:** Primary runtime environment.
- **Git:** For version control.
