# Olympic Data is a simple UI interface to explore data about Olympic athletes and data correlation between several variables like height,weight, country, sport, etc.
key advantages of Olympic:

* Simple interface to provide insightful data
* Stacked filtered system that provides a way to iterative drill down data based on data attributes.
* Nearly instant response
